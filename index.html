# 核心技术栈
- 界面框架: tkinter
- 网络请求: requests
- 音频播放: pygame
- 数据解析: json, re
- 多线程: threading
- 文件处理: os, tempfile

# API调用示例
def get_song_url(song_id):
    try:
        url = f"https://music.163.com/song/media/outer/url?id={song_id}.mp3"
        return url
    except Exception as e:
        print(f"获取歌曲URL失败: {str(e)}")
        return None

常见问题

解答你可能遇到的问题

为什么有些VIP音乐无法播放？

由于网易云音乐API的权限限制和版权保护机制，目前版本中部分VIP专属音乐暂时无法播放。这是公共API的限制，我们正在研究可能的解决方案。

五花肉云音乐支持哪些操作系统？

目前支持Windows 10/11系统。由于是基于Python开发，理论上也可以在Linux系统上运行，需要手动安装相关依赖。macOS版本正在开发中，敬请期待。

如何更新到最新版本？

公测版目前需要手动更新。我们会在官网和GitHub仓库发布最新版本，你可以下载新版本并覆盖安装。未来我们会加入自动更新功能。

下载的音乐保存在哪里？

你可以在下载时选择保存路径。默认情况下，音乐文件会保存在用户目录下的"五花肉云音乐"文件夹中。你也可以在设置中更改默认下载位置。

如何反馈问题或建议？
